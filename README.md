# 🐍 Snake Pro Max

A modern Snake Game built using HTML, CSS, and JavaScript with advanced features.

## 🚀 Features
- Start Screen UI
- Pause Functionality
- Restart Option
- Score & High Score System
- Difficulty Levels

## 🛠️ Technologies Used
- HTML
- CSS (Grid + Animations)
- JavaScript (OOP + Game Logic)

## 🎮 Controls
- Arrow Keys → Move Snake
- Enter → Restart Game
- Pause Button → Pause Game
